จุดประสงค์ของโปรเจกต์นี้คือ "การทำนายว่าลูกค้าคนใดมีแนวโน้มที่จะซื้อจักรยาน (Purchased Bike = Yes/No)" โดยศึกษาและวิเคราะห์จากตัวแปรปัจจัยต่าง ๆ ของลูกค้า เช่น รายได้ (Income), อายุ (Age), จำนวนรถยนต์ที่มี (Cars), และสถานะสมรส (Marital Status) เพื่อนำข้อมูลมาสร้างโมเดล Machine Learning ในการคัดแยกกลุ่มลูกค้าเป้าหมายสำหรับทำการตลาดได้อย่างแม่นยำ
โดยมีขั้นตอนการดำเนินงาน ดังนี้ 
- ทำความสะอาดข้อมูลที่มีค่าสูญหาย (Missing Values) โดยใช้ค่า Median และ Mode แทนที่ช่องว่างให้สมบูรณ์ พร้อมตัดข้อมูลที่ซ้ำซ้อนออก
- แปลงข้อมูลประเภทตัวอักษรให้เป็นตัวเลขด้วย LabelEncoder และแบ่งข้อมูลออกเป็นชุด Train 80% สำหรับสอน AI และ Test 20% สำหรับวัดผลความแม่นยำ
- พัฒนาและเปรียบเทียบประสิทธิภาพของโมเดล Machine Learning 2 ตัว คือ Logistic Regression และ Random Forest
ผลลัพธ์จากกราฟ (Data Insights)
จากการวิเคราะห์ตารางสหสัมพันธ์ (Correlation) พบพฤติกรรมของข้อมูลที่น่าสนใจ เช่น กลุ่มที่มีรายได้สูง (Income) จะมีแนวโน้มการครอบครองรถยนต์ (Cars) ที่สูงตามไปด้วย 
และจากการทดสอบกับข้อมูลชุด Test ผลลัพธ์ยืนยันว่า โมเดล Random Forest มีความแม่นยำ (Accuracy) สูงกว่า Logistic Regression เนื่องจากมีความสามารถในการจัดการกับเงื่อนไขข้อมูลพฤติกรรมที่ซับซ้อนได้ดีกว่า
